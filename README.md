# LAB_Microsoft_Azure_Computacao_em_Nuvem
# ☁️ Resumo de Conceitos de Computação em Nuvem

## 🛠 Pré-requisitos de Computação em Nuvem
- Noções básicas de redes, sistemas operacionais e segurança.
- Conhecimento sobre virtualização.
- Entendimento dos modelos de serviços em nuvem.

## ☁️ Conceitos Fundamentais
- **Elasticidade:** ajuste dinâmico de recursos conforme a necessidade.
- **Escalabilidade:** expansão da capacidade sem perda de desempenho.
- **Cobrança sob demanda:** pagamento somente pelo que for utilizado.
- **Self-service:** usuários podem provisionar recursos sem intervenção manual.

## 🏢 On-Premise vs Cloud
- **On-Premise:** infraestrutura local gerida internamente.
- **Cloud:** serviços hospedados remotamente por provedores, com menor custo inicial e maior flexibilidade.

## ⚙️ Modelo Híbrido
- Integra recursos locais e em nuvem.
- Ideal para cenários com dados sensíveis ou exigências legais.

## 🌍 Tipos de Regiões
- Regiões geográficas onde os datacenters estão localizados.
- Influenciam na latência e no cumprimento de regras de soberania de dados.

## 📦 Modelos de Serviço em Nuvem

| Modelo | Significado | Exemplos |
|--------|-------------|----------|
| **IaaS** | Infraestrutura como Serviço | Máquinas Virtuais |
| **PaaS** | Plataforma como Serviço | Azure App Services |
| **SaaS** | Software como Serviço | Microsoft 365 |

## 🔧 Serviços da Azure
- **Redundância:** alta disponibilidade e recuperação de falhas.
- **Azure Data Box:** transferência física de grandes volumes de dados.
- **Configuração de rede:** redes virtuais, VPNs, balanceamento de carga.

## 🧪 Criando uma Conta Trial na Azure
1. Acesse [azure.microsoft.com](https://azure.microsoft.com/)
2. Clique em **“Iniciar gratuitamente”**.
3. Registre-se com um e-mail válido e uma forma de pagamento (sem cobrança).
4. Ganhe créditos gratuitos para testes (ex: R$1.000 ou US$200).
5. Utilize os recursos disponíveis por até 30 dias para práticas de laboratório.
